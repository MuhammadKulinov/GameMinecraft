# Текст для README.md
Прыгающий желтый шарик в Майнкрафте
Это моя мини-игра на Питоне. Главная задача программы — построить полосу препятствий в Майнкрафте и запустить по ней желтый шарик (блок), который сам умеет перепрыгивать преграды и добираться до финиша.

# Что делает код:
Строит длинную дорожку из белых блоков.

Расставляет красные блоки — это препятствия.

Запускает желтый блок (шарик), который едет вперед.

Когда шарик «видит» перед собой красную преграду, он автоматически подпрыгивает, перелетает её и едет дальше.

В конце шарик доезжает до зеленого блока и пишет в чат, что игра пройдена.

# Как запустить:
Зайти на сервер Майнкрафт.

Установить библиотеку mcpi через терминал. Она нужна, чтобы Питон мог управлять блоками в игре. Команда для установки:
pip install mcpi

Запустить файл main.py.
