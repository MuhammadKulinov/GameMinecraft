# 🏎️ Bild.py — Аркадные гонки в Minecraft
Bild.py — это динамичная гоночная игра, реализованная на Python специально для мира Minecraft. Забудь про сложную установку модов: проект поставляется со встроенным локальным сервером, поэтому всё работает «из коробки». Почувствуй драйв классических аркад прямо в кубической вселенной!

# 🌟 Ключевые особенности проекта
Полная автономность: В папке уже есть настроенное ядро сервера Minecraft.

Умный трафик: Машины под управлением ИИ сканируют дорогу и умеют тормозить, чтобы избежать аварий.

Эффекты Nitro: Ускорение сопровождается сменой внешнего вида машины и визуальным следом из редстоуна.

Динамичный мир: Трасса, деревья и освещение генерируются в реальном времени прямо вокруг игрока.

Прогрессирующая сложность: Чем дольше ты играешь, тем быстрее появляются новые препятствия.

#📁 Что внутри репозитория
server — готовое ядро сервера Minecraft с настроенными конфигами.

Start_Server.lnk — ярлык для мгновенного запуска серверной части.

race.py — «сердце» игры: здесь живет вся логика, физика и правила.

input_system.py — модуль, который отвечает за мгновенную реакцию на нажатия клавиш.

mcpi — библиотека для прямой связи твоего кода с миром игры.

requirements.txt — список необходимых дополнений для Python.

#💻 Системные требования
Игра: Minecraft Java Edition (рекомендуемая версия 1.16.5).

Язык: Python версии 3.7 или выше.

ОС: Windows (для работы стартового файла сервера).

#🚀 Как запустить игру
Запуск сервера: Открой файл Start_Server.lnk. Подожди, пока в черном окне консоли появится надпись о готовности мира.

Вход в Minecraft: Запусти игру и подключись по адресу localhost.

Установка библиотек: В терминале своего редактора (например, VS Code) напиши:

pip install -r requirements.txt

Старт: Запусти главный скрипт командой python race.py.

Трасса автоматически построится вокруг тебя — гонка началась!

#🎮 Управление
Стрелки Влево / Вправо — Маневрирование между полосами движения.

Стрелка Вверх — Активация Nitro. Ты получишь ускорение х2 на 3 секунды, а монеты будут давать в два раза больше очков! (Перезарядка — 8 секунд).

Клавиша Esc — Мгновенная остановка игры и выход.

#⚙️ Игровые механики
Твой транспорт и противники
В игре три типа машин, и у каждой свои правила:

Мотоцикл: Маленький и быстрый. За его обгон ты получишь 20 очков.

Легковой автомобиль: Средний по всем параметрам. Дает 10 очков.

Грузовик: Настоящий гигант. Занимает две полосы, едет медленно, дает 5 очков.

Жизни и бонусы
Запас здоровья: У тебя есть 3 жизни. Столкновение с машиной забирает одну жизнь, но убирает препятствие с пути.

Золотые монеты: Собирай их, чтобы получить +50 очков к итоговому счету.

#🛠 Настройка под себя
Если ты хочешь сделать игру сложнее или проще, открой начало файла race.py. Там можно изменить:

ROAD_WIDTH — количество полос на дороге.

BASE_GAME_SPEED — начальную скорость игры.

SPAWN_POOL — шанс появления тех или иных типов машин.
